# SauceDemo E2E & API Testing Portfolio

Bu layihədə **SauceDemo** tətbiqinin uçdan-uca (E2E) testi **Cypress** vasitəsilə avtomatlaşdırılmışdır. Bundan əlavə, **Postman** vasitəsilə **JSONPlaceholder** API-si üzərində müxtəlif sorğular (POST, GET, PUT, PATCH) icra edilmiş, performans testləri keçirilmiş və nəticələr hesabatlaşdırılmışdır.

## Texnologiyalar
- **Automation:** Cypress
- **API Testing:** Postman
- **Documentation:** Excel & GitHub
